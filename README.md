# Afrida's Personal Portfolio

A personal portfolio template built with **Next.js 15** and **TypeScript**, styled with an aesthetic — bold borders, hard shadows, and a punchy color palette.

**Live demo:** [afridarn.github.io/personal-portfolio](https://afridarn.github.io/personal-portfolio)

![Next.js](https://img.shields.io/badge/Next.js-15-black?style=flat-square)

![License](https://img.shields.io/badge/license-MIT-green?style=flat-square)

## Features

- Neobrutalism design with thick borders and hard offset shadows
- 6 built-in color palettes, switchable in one line
- All content (experience, projects, skills, contact) driven from a single JSON file

## Tech Stack

- [Next.js 15](https://nextjs.org/) (App Router, static export)
- TypeScript
- CSS custom properties — no CSS framework
- GitHub Actions for CI/CD

---

## Getting Started

```bash
git clone https://github.com/afridarn/personal-portfolio.git
cd portfolio
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
